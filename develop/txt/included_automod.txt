[hh]AutoMOD[/hh]

AutoMOD is a tool designed to parse and automatically install MODX MODifications for phpBB3.
It also has the ability to uninstall MODs.
It is a useful tool for MOD authors and end users alike: All MODs are installed with AutoMOD prior to being accepted in the MODs Database. As such, any MOD listed in the MOD Database should install with AutoMOD.

[b]Included version:[/b] 1.0.0 RC 3

[b]Available at:[/b] [url]http://www.phpbb.com/mods/automod/[/url]
