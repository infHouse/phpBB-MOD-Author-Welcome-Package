Introduction

Hello there!


I am Obsidian, and I have been working with modifying phpBB for about two years now. I'm writing this to help get you started with learning to build MODs for phpBB 3.0.x (codenamed "Olympus"), and to provide you with the resources and tools you need to succeed.

Not sure what a phpBB MOD is?


Well, I'll explain.
Because of phpBB being an open-source project, you are free to modify it to suit your needs as you wish. When you modify your copy of phpBB, document the changes you have made, and distribute those changes, in essence you have released a MOD.
In reality, it is a bit more involved than that, but don't sweat it; we'll get to that later.

What all is in this welcome package?


This welcome package contains the guide that you are reading now, and several tools that are used by many MOD authors that you may want to use yourself. A more in-depth overview of the tools included will appear later on in this guide.

What will this guide cover?


Actually, it will be a broad range of things, including:

  • The development process

  • Tools of the trade

  • Coding and development resources

  • Utilizing built-in phpBB3 functionality

  • Humorous source code comments in phpBB3
